# Claude-Mem 使用文档

## 📖 简介

Claude-Mem 是一个为 Claude Code 设计的持久化记忆系统，能够自动记录你的工作历史，并在未来的会话中自动加载相关记忆。

## 🎯 核心功能

- **持久化记忆**：所有会话记录永久保存在本地数据库
- **自动工作**：无需手动操作，自动记录和加载
- **智能搜索**：使用自然语言搜索历史工作记录
- **Web 界面**：通过浏览器查看所有历史记忆
- **隐私保护**：使用 `<private>` 标签排除敏感内容

## 🚀 快速开始

### 1. 安装 Bun（已完成）

```bash
# 验证 Bun 是否安装成功
bun --version
```

### 2. 启动 Worker 服务

```bash
cd "F:/工作/码码乐/工作—码码乐-开发/code_tools/libraries/claude-mem"
npm run worker:start
```

### 3. 查看服务状态


输出示例：
```

  Port: 37777
  Started: 2026-03-03T07:30:49.195Z
```

### 4. 访问 Web 界面

打开浏览器访问：http://localhost:37777

## 📊 工作原理

### 自动记录流程

1. **SessionStart**：会话开始时，自动加载相关历史记忆
2. **UserPromptSubmit**：用户提交提示词时记录
3. **PostToolUse**：每次工具使用后记录观察结果
4. **SessionEnd**：会话结束时生成总结并保存

### 数据存储位置

- **SQLite 数据库**：`C:\Users\Administrator\.claude-mem\claude-mem.db`
- **Chroma 向量库**：`C:\Users\Administrator\.claude-mem\chroma\`
- **配置文件**：`C:\Users\Administrator\.claude-mem\settings.json`
- **日志文件**：`C:\Users\Administrator\.claude-mem\logs\`

## 🔒 隐私保护

### 排除敏感内容

使用 `<private>` 标签包裹敏感信息：

```
<private>
API_KEY=sk-1234567890abcdef
PASSWORD=my_secret_password
</private>
```

被标记的内容不会被存储到数据库中。

## ⚙️ 配置文件

配置文件位置：`C:\Users\Administrator\.claude-mem\settings.json`

主要配置项：

```json
{
  "port": 37777,
  "enableChroma": true,
  "maxObservations": 1000,
  "compressionLevel": "medium"
}
```

## 🐛 故障排除

### Worker 服务无法启动

1. 检查端口是否被占用：
```bash
netstat -ano | findstr :37777
```

2. 查看日志：
```bash
npm run worker:logs
```

### 没有记录任何数据

1. 确认 Worker 服务正在运行：
```bash
npm run worker:status
```

2. 确认插件已正确安装到 Claude Code

3. 重启 Claude Code

### Bun 命令找不到

1. 重启终端
2. 验证环境变量：
```bash
echo $PATH | grep bun
```
