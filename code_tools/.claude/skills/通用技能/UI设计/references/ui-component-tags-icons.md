# 标签与图标规范

## 成员/状态标签

- 展示短小附加信息（成员姓名、状态）
- 内容简短，可通过颜色/图标区分类别
- 同一类型颜色含义保持一致
- 支持关闭/编辑时有明显控件

## 图标使用原则

- 图标用于加强识别效率，不替代文字说明
- 重要操作或状态同时提供图标和文字
- 相同含义图标全站保持一致
- 不同含义避免使用相似图形

## 高亮提醒区域

- 展示重要提醒、警告信息
- 同一页面尽量少量出现
- 包含：清晰标题 + 说明文字 + 下一步行动
- 动画节奏平稳，不干扰阅读

## 社交登录区域

- 统一放在一个容器中
- 层级低于主登录方式
- 图标+简短文字（"Google"、"Apple ID"）
- 分组上方有引导文案
