# 组件设计规范概览

本规范只约束「组件是什么」「负责什么」「如何交互」，不规定具体视觉风格。

## 核心通则

- **主按钮**：每视图只有一个，有明确文字标签
- **卡片**：结构一致（标题区+内容区+操作区）
- **交互**：状态变化即时反馈
- **图标**：全站含义一致，不替代文字
