# 背景设计规范

背景是营造氛围感的关键，使用柔和渐变。

---

## 渐变类型

### 同色渐变

```css
background: linear-gradient(180deg, 
  浅色 0%,
  中色 50%,
  白色 100%
);
```

### 多色渐变

```css
background: linear-gradient(135deg,
  色彩1 0%,
  色彩2 50%,
  白色/米白 100%
);
```

### 弥散渐变

```css
background: linear-gradient(
  120deg,
  rgba(色1, 0.3) 0%,
  rgba(色2, 0.2) 50%,
  rgba(色3, 0.3) 100%
);
```

---

## 背景规范

- **过渡柔和**：渐变过渡平缓，避免色彩突变
- **低饱和度**：背景颜色饱和度低，不抢前景内容
- **轻盈感**：背景趋向白色/米白，营造漂浮感
- **可选元素**：极淡的几何图形、光斑、噪点纹理
