# 前端界面设计指南

> 目的：创建独特、生产级的前端界面，避免通用AI美学。

## 设计思考流程

1. **目的**：界面要解决什么问题？谁在使用？
2. **风格**：选一个明确方向并保持一致
   - 极简/极繁/复古未来/有机自然/奢华精致
   - 玩具感/杂志感/粗野主义/几何装饰/柔和粉彩
3. **记忆点**：一个让人记住的独特特征
4. **约束**：技术栈、性能、响应式

## 美学指南

**字体**：选择独特有趣的字体，避免Arial/Inter等通用字体

**配色**：承诺一个连贯的美学，用CSS变量管理，主色大胆、强调色犀利

**动效**：重点时刻动画（首屏入场、滚动触发、悬浮惊喜）

**布局**：非对称、重叠、对角线流动、破格元素、大留白或有控制的密度

**背景**：营造氛围和深度，渐变网格/噪点纹理/几何图案/透明叠层/戏剧阴影

## 避免AI通用风

❌ **禁止**：

- 通用字体（Inter、Roboto、Arial、系统字体）
- 紫色渐变+白底的老套配色
- 可预测的布局和组件模式
- 缺乏个性的模板化设计

✅ **要求**：

- 创意解读，做出意料之外的选择
- 每个设计都不同，变换明暗主题、字体、美学
- 复杂度匹配愿景：极繁需要精心动效，极简需要精确克制
