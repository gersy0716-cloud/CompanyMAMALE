# 画布设计 (Canvas Design)

创建博物馆级的视觉艺术作品。核心是**视觉哲学先行，图形表达在后**。

## 🎨 创作流程

1. **构思视觉哲学**：撰写 4-6 段关于空间、形体、色彩与节奏的“视觉宣言”。
2. **提炼概念线索**：识别用户请求中的细微概念（而非字面意思），将其作为设计的 DNA。
3. **视觉表达**：通过几何图形、重复模式、负空间利用等手段表现哲学。
4. **极致工艺**：对齐精度、材质纹理、排版节奏的反复推敲。

## ⚖️ 核心原则

- **视觉哲学**：不仅是布局，而是一种审美世界观。
- **极简文字**：文字是视觉元素的一部分。通常使用极细字体、临床式标签。
- **空间表达**：通过重力感、张力、比例和节奏传递信息，而非段落描述。
- **专家级工艺**：看起来像是顶尖设计师花费数十小时打磨的作品。

## 📚 视觉哲学示例

- **Concrete Poetry (混凝土诗歌)**：通过巨大的色块、雕塑般的字体和波兰海报式的空间布局传达力量。
- **Chromatic Language (色彩语言)**：色彩作为核心信息系统。极简排版，让色场（Chromatic Fields）说话。
- **Analog Meditation (模拟冥想)**：纸张纹理、墨渍、大量负空间。日本摄影集审美。

## ✅ 验收标准

- [ ] 是否完成了 4-6 段的视觉哲学撰写？
- [ ] 文字是否足够精简？是否存在大段解析？
- [ ] 间距、边距和对齐是否达到完美精度？
- [ ] 整体审美是否具有系统性的观察感（类似科学图表或艺术珍藏）？
