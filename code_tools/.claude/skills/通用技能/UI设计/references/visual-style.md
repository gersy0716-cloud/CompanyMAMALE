# 核心视觉风格

**关键词：** `清透` `柔光` `空气感` `低饱和`

---

## 质感原理

- **清透感：** 半透明、柔光、光晕扩散
- **色彩基调：** 低饱和度、高亮度
- **背景氛围：** 渐变过渡，轻盈不厚重
- **视觉隐喻：** 玻璃、气泡、柔和光影

**❌ 避免：** 高饱和、纯色块、强烈阴影、金属质感、锐角

---

## 配色使用

- **同色渐变**：同一色系的深浅变化
- **多色渐变**：2-3种颜色的过渡组合
- **弥散渐变**：梦幻朦胧的色彩过渡
- **对比配色**：主色+强调色组合

> 背景保持浅色，趋向白色/米白，营造轻盈感

---

## 氛围与情绪

**目标感觉：** 清新 | 明亮 | 氛围感 | 透明柔光 | 高级 | 舒适无压迫

---

# 排版与空间

高质量 UI 的秘诀在于呼吸感与秩序感。

## 文本排版

- **行宽限制**：段落宽度应控制在 **500px - 700px**
- **视觉阶梯**：使用阶梯比例（如 1.25x），40px 标题 → 32px 副标题
- **对比强度**：
  - 标题：Strong font-weight，高对比度
  - 副标题：低对比度或 Medium weight
- **对齐保障**：多行文本坚持**左对齐**，居中仅用于 1-2 行

## 空间与嵌套

- **负空间优先**：减少分割线，通过 Whitespace 实现"隐形分组"
- **2x 圆角法则**：外半径 = 内半径 * 2
- **填充平衡**：卡片内含非圆形元素时，边缘 Padding 设为双倍

# 材质与质感

所有组件：**半透明 + 微微发光**

## 玻璃拟态

- **模糊度：** 10–30px
- **透明层：** 10–20% 白色叠加
- **使用场景：** 重要卡片、悬浮元素、弹窗

```css
background: rgba(255, 255, 255, 0.7);
backdrop-filter: blur(12px);
border: 1px solid rgba(背景色, 0.8);
box-shadow: 0 8px 32px rgba(主色, 0.15);
```

## 新拟态 (Neumorphism)

- **双向阴影：** 内陷+外凸
- **按下内陷效果**
- **使用场景：** 返回、取消、次要操作按钮

```css
background: 背景色;
box-shadow: 
    6px 6px 12px rgba(主色, 0.2),
    -6px -6px 12px rgba(255, 255, 255, 0.9);

:active {
    box-shadow: 
        inset 4px 4px 8px rgba(主色, 0.25),
        inset -4px -4px 8px rgba(255, 255, 255, 0.8);
}
```

## 圆角规范

| 元素 | 圆角 |
|------|------|
| 卡片 | 16-20px |
| 按钮 | 12-16px |
| 小标签 | 10-12px |

**公式：** 外圆角 = 内圆角 + padding(4px)

## 阴影规范

```css
/* 使用环境色阴影 */
box-shadow: 
  0 8px 32px rgba(环境色, 0.12),
  0 2px 8px rgba(环境色, 0.08);
```

❌ 避免深灰/黑色阴影
