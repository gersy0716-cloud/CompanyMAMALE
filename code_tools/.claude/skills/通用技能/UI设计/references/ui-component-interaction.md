# 交互组件规范

## 开关切换（Toggle）

- 用于二元状态切换（开/关）
- 左右两侧有明确文案说明当前状态
- 状态变化立即生效或有"保存"按钮
- 键盘可用空格/回车触发

## 进度条（Progress）

- 优先使用确定进度（百分比/剩余项数）
- 附近显示文字说明："正在导入 3/10 条记录"
- 失败时停止动画并显示错误提示

## 拖拽区域（Drag & Drop）

- 明确边界和提示文案
- 拖入/离开/释放时提供视觉反馈
- 拖拽完成后重新渲染源列表
- 避免已拖入项重复显示

## 导航菜单

- 简短易懂的文字标签+语义清晰的图标
- 当前活跃项有明显状态反馈
- 悬停时有交互反馈但不改变"当前所在页面"标记
- 窄屏可折叠为抽屉/汉堡菜单
