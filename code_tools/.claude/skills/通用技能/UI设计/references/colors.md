# 配色方案库

> **设计原则:** 单一色系，通过颜色渲染和深浅变化营造层次感

---

## 快速选择

| 情绪 | 推荐色系 |
|------|----------|
| 清新自然 | 绿色系 |
| 专业沉稳 | 蓝色系、中性系 |
| 甜美柔和 | 粉色系 |
| 活力温暖 | 橙黄系 |
| 梦幻浪漫 | 渐变系 |
