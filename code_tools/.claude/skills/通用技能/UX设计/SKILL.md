# UX 设计规范 (Knowledge Base)

> 💡 用户体验 | 🖱️ 交互逻辑 | 🧠 心理引导 | 📱 响应式设计

本技能旨在通过深度行为分析与心理学规律，最小化用户认知负担，提升产品转化率与操作流畅度。

## 🎯 核心体验准则

- **减少摩擦 (Frictionless)**：前置解决由于环境或逻辑产生的阻力（如在点击前预检权限）。
- **反馈及时**：异步操作必须有即时的状态响应，危险操作必须有明确的影响告知。
- **一致性**：全站交互模式、提示语风格与反馈节奏必须保持逻辑统一。
- **感知引导**：利用心理锚点（如 30 秒上手）降低用户决策门槛。
