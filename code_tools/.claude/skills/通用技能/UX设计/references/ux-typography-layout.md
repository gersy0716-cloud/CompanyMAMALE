# 排版布局规范 (Typography & Layout)

## 间距体系 ⭐

**4px 基准间距尺度：**
`4px / 8px / 12px / 16px / 20px / 24px / 32px / 48px`

**常用场景：**

- 组件内元素：8px, 12px
- 组件之间：16px, 24px
- 区块之间：32px, 48px
- 页面边距：16px（移动）, 24-32px（桌面）

## 行距规范 ⭐

```
标题 (Headings):    1.2–1.3
正文 (Body):        1.5–1.6
按钮/标签:          1.0–1.2
段落间距:           16px / 24px
```

## 布局原则

- **层级顺序：** 图片 → 标题 → 描述 → 标签 → 按钮
- **严格对齐：** 左对齐或居中
- **明确分组：** 组与组之间明显区隔

### 古腾堡原则与 Z 型视觉流

- 用户多按 Z 型路径浏览（左上 → 右上 → 左下 → 右下）。
- 在 Z 型节点放置重要信息：左上 Logo/导航，右上辅助，左下说明，**右下核心 CTA**。
- 视觉权重应与浏览路径匹配，避免在死角隐藏关键操作。

## 教师/学生端布局偏好

- **小组管理**: 优先垂直布局（flex-direction:column），确保 100% 宽度可读性。
- **UI 架构**: 学生端隐藏导航，直达内容；教师端班级选择作为入口，选后进入管理界面。

## 文字排版准则

- **字体**: `Inter`, `SF Pro`, `PingFang`, `Microsoft YaHei`, `Source Han Sans`
- **字重**: 标题 Medium/Semibold | 正文 Regular
- **间距**: 字间距轻微增加以提升呼吸感。
- **禁止**: 过粗字体、荧光渐变或滥用衬线体。
