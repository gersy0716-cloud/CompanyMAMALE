# 心理学与引导 (UX Psychology)

通过理解人类认知的偏见与习惯，设计更具转化力的交互。

## 🧠 核心心理法则

- **峰终定律 (Peak-End Rule)**：用户对体验的评价由“峰值（最爽/最痛）”与“结尾”决定。
  - **技巧**：在流程结束或关键成就点埋入“惊喜”动画或文案。
- **人脸的力量 (Power of Faces)**：人类本能地关注人脸。
  - **建立信任**：使用真实的人像图片远比抽象图标更易建立情感连接。
  - **视线引导 (Gaze Following)**：让图片中人物的视线指向你的主按钮 (CTA)。
- **决策简化 (Analysis Paralysis)**：
  - **限量选择**：定价方案建议限制在 3-4 个。
  - **中间项效应**：通过视觉高亮（阴影、边框）引导用户选择“最受欢迎”项。

## 📈 转化引导技巧

- **价值计算器 (ROI Calculators)**：与其说“省钱”，不如提供一个交互式计算器，让用户算出具体数字。
- **数字的力量**：
  - **具体化**：用“已有 12,400+ 开发者加入”代替“深受好评”。
  - **证言增强**：在客户证言中**加粗核心短语**（如“节省了 **40%** 时间”），确保扫描时也能抓住重点。
- **微评价 (Micro-testimonials)**：在阻力最大的点（如支付页、注册表单底端）放置一条简短强力的真实评价。
- **异议即时处理 (Objection Handling)**：在主 CTA 按钮旁标注“无需信用卡”或“免费试用 14 天”，消除最后决策阻力。
- **价值预览**：在主按钮旁标注“无需信用卡”或“30秒上手”，消除最后的疑虑。

## 🛠️ 实战引导准则

- **空状态模板**：空状态不要只给一个按钮，提供 2-3 个预设模板让用户一键进入状态。
- **迁移成本**：显式展示“一键迁移”工具，降低用户切换平台的心理门槛。
