# 页面布局模式 (Page Layouts)

## 视觉特征 (Visual Characteristics)

- **卡片层叠**：毛玻璃卡片（Glassmorphism）浮于渐变大背景之上。
- **呼吸感留白**：边距保持在 24-32px。
- **浮动容器**：圆角统一 20px，配合柔和环境色阴影。

## 常见布局模型

### 居中核心卡片

(适用于仪表盘、个人中心、登录页)
主卡片居中承载核心逻辑，次要卡片围绕或平铺在下方。主内容占屏控制在 30-50%。

### 网格动态平铺

(适用于功能入口、学生名册、资源列表)
移动端保持单列展示，桌面端根据内容密度控制在 3-4 列。

## 空间与层级规范 (Hierarchy)

- **Z 轴深度**：背景层 → 次级卡片层 → 主操作卡片 → 悬浮 CTA/通知层。
- **危险区域 (Danger Zone)**：红色边框或背景严格仅用于不可逆行为（删除、解绑）；普通警告避免用红。
- **视觉锚点**：在多方案对比中，通过阴影提升 (Elevation) 或微弱发光高亮推荐选项。
