# Ant Design 组件库

> 企业级 UI 设计语言和 React UI 库

## 📦 安装

```bash
npm install antd
```

## 🔨 快速使用

```jsx
import { Button, DatePicker, ConfigProvider } from 'antd';

// 主题定制
<ConfigProvider
  theme={{
    token: {
      colorPrimary: '#00b96b',
      borderRadius: 8,
    },
  }}
>
  <Button type="primary">主要按钮</Button>
</ConfigProvider>
```

## 🧩 组件分类

- **通用**: Button, Icon, Typography
- **布局**: Grid, Layout, Space, Flex
- **导航**: Tabs, Steps, Dropdown, Menu
- **录入**: Form, Input, Select, DatePicker, Upload
- **展示**: Table, Card, List, Collapse, Calendar
- **反馈**: Modal, message, Drawer, Skeleton, Spin
