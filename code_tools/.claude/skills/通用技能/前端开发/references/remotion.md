# Remotion 视频编程

> 使用 React 编程式创建视频的框架。

---

## 核心 Hook

- **useCurrentFrame()**: 获取当前帧。
- **useVideoConfig()**: 获取 fps, duration, width, height。

---

## 动画函数

- **interpolate()**: 值映射与补间。
- **spring()**: 物理弹簧效果。
- **interpolateColors()**: 颜色过渡。

---

## 核心组件

- **<Composition>**: 视频基本配置。
- **<Sequence>**: 时间线片段控制。
- **<Series>**: 顺序排列片段。
- **<AbsoluteFill>**: 全屏容器。

---

## 媒体处理

- **<OffthreadVideo>**: 高性能视频渲染。
- **<Audio>**: 音频轨道。
- **staticFile()**: 载入项目静态资源。

---
