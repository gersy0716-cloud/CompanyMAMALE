# 学习搭子智能体 

**对话式学习：AI 扮演特定角色引导深度对话，培养批判性思维与逻辑推理能力**

## 一、产品定位

学习搭子是面向学生与教师的轻量化智能学习辅助工具：

- **学生端**：提供“随便问问”的陪伴式答疑与“专项提升”的深度练习，支持**难点汇总、自选薄弱点出题、学习进度可视化**。
- **教师端**：提供**班级共性学习问题汇总、学生薄弱点概况查看**，辅助精准教学。

## 二、解决的核心问题

### 学生侧

1.  **提问场景杂乱**：日常作业疑问涉及多学科，缺乏统一且专业的解答入口。
2.  **学习无方向**：不清楚自己高频薄弱、易错的知识点。
3.  **刷题被动**：被动接受强制推送的练习，缺乏针对性与主动性。
4.  **复盘困难**：试卷、笔记等文件类疑问无法快速解析并沉淀为知识资产。

### 教师侧

1.  **学情模糊**：难以快速掌握班级整体集中的薄弱、易错知识点。
2.  **效率低下**：人工统计学生问题费时费力，备课缺乏数据参考。
3.  **辅导粗放**：无法简洁获取单个学生核心薄弱点，难以进行针对性辅导。

## 三、核心价值

-   **学生**：分场景答疑（闲聊/专攻）、清晰定位薄弱点、直观看到进步。
-   **教师**：快速获取班级共性问题，辅助课堂重点讲解，提升教学效率。

## 四、学生端功能

### 1. 基础信息与场景
 **基础信息同步**：系统自动读取账号年级信息（如：七年级），作为底层对话上下文。

 **频道式切换设计**： 

- **默认状态（随便问问）**：在输入框上方或顶部，默认选中“全能助手”频道。此时支持多学科混杂提问、百科问答，AI 使用通用模型回复，**不强制打标**。    

- **一键切换（专项频道）**：用户点击“数学”、“英语”等学科图标/标签，界面微动效提示（如主题色变更），即刻进入**专项深度模式**。    

* **深度模式逻辑**：选中特定学科后，AI 自动加载该学科专属 System Prompt（如数学老师的逻辑引导），并对后续对话进行**严谨的知识点拆解与后台记录**。

### 2. 智能答疑与解析
* **多模态交互**：支持语音 / 文字实时提问。
* **文件解析**：支持上传学习文件（试卷、笔记图片等），系统基于**后台提示词（Prompt）**自动识别并解析内容，同步提取知识点与薄弱点（不直接给出答案，而是引导思路）。
* **思路笔记**：答疑、练习后自动生成轻量化思路笔记，支持查看与回顾。

### 3. 难点攻克闭环
* **难点看板**：自动统计并汇总个人高频难点，展示为简洁的“待攻克”面板。
* **自主靶向出题**：学生可手动选择面板中任意难点，发起针对性出题练习。
* **难度分层**：出题前可自主选择难度：**基础 / 进阶 / 综合**。
* **自适应生成**：根据所选难点、难度及年级上下文，生成对应练习题，支持自主在线作答。

### 4. 进度与反馈
* **实时反馈**：答题与复盘后，自动更新对应难点的掌握度与进度，并展示提升效果。
* **学习总览**：展示个人学习全景：已解决难点、待解决痛点、各知识点掌握情况。
* **针对性复盘**：支持手动针对指定难点发起二次复盘巩固。
* **错题消消乐”机制 (Gamified Review)**：
  - **触发场景（记忆回调）**：
    - 在“随便问问”的闲聊间隙，AI 基于**艾宾浩斯遗忘曲线**发起“无压力的知识回溯”。
  - **交互逻辑（情境化追问）**：
    - **拒绝生硬弹窗**：AI 不会直接甩出一张试卷，而是以“聊天”的方式切入。
    - **话术示例**：“对了，咱们上周在**[勾股定理]**上卡了一下。我刚看到一个挺有意思的梯子靠墙问题，你要不要试着拆解一下看看？”
    - **变式训练**：如果学生接话，AI 顺势抛出一道基于历史错题改编的**应用场景题**（变式题）。
  - **激励反馈**：
    - 学生只需完成**思路拆解**或**得出正确答案**，“难点看板”中对应的薄弱点图标即产生消除特效或点亮星星，给予即时的成就感反馈。

### 5. 数据权限
* 仅展示本人学习数据，无法查看其他学生信息。
* 支持手机、平板、电脑多端使用。

## 五、教师端功能

1.  **班级学情大盘**：自动汇总所带班级的高频易错点、共性薄弱知识点，形成清晰清单（如：全班60%学生在“函数”章节有记录）。
2.  **知识点透视**：支持按知识点筛选，查看班级在该点上的集中问题分布。
3.  **学生画像概览**：可查看单个学生的核心薄弱点概况（仅展示知识点标签列表，无细粒度聊天记录，保护学生隐私）。
4.  **辅助备课与资源生成**：
    * **智能同类题推荐**：教师点击某个共性薄弱点（如“浮力计算”），AI 自动推荐 3-5 道**同类型的典型易错题**（含解析）。
    * **讲义导出**：支持将“学情清单”与“推荐题目”一键导出为备课参考文档，直接用于课堂重点讲解。
5.  **权限控制**：
    * 仅提供数据查看与导出，无任何对学生端的直接操作、推送或干预权限。

# 学习搭子智能体 - 研发需求说明 (v2.4)

**对话式学习：AI 扮演特定角色引导深度对话，培养批判性思维与逻辑推理能力**

## 一、产品定位

学习搭子是面向学生与教师的轻量化智能学习辅助工具，支持**独立使用**或作为**桌面/浏览器浮窗**挂载：

- **学生端**：提供“随便问问”的陪伴式答疑与“专项提升”的深度练习，支持**难点汇总、自选薄弱点出题、学习进度可视化**。
- **教师端**：提供**班级共性学习问题汇总、学生薄弱点概况查看**，辅助精准教学。

## 二、解决的核心问题

### 学生侧

1.  **提问场景杂乱**：日常作业疑问涉及多学科，缺乏统一且专业的解答入口。
2.  **交互繁琐**：在网页/网课学习时，遇到不懂的内容需要截图、保存、上传，操作路径长。
3.  **学习无方向**：不清楚自己高频薄弱、易错的知识点。
4.  **复习枯燥**：被动接受刷题，缺乏趣味性和针对性的考前突击手段。

### 教师侧

1.  **学情模糊**：难以快速掌握班级整体集中的薄弱、易错知识点。
2.  **备课耗时**：人工查找同类易错题进行针对性训练非常费时。
3.  **辅导粗放**：无法简洁获取单个学生核心薄弱点，难以进行针对性辅导。

## 三、核心价值

-   **学生**：沉浸式伴随答疑、游戏化复习、考前精准冲刺。
-   **教师**：数据驱动教学、一键生成针对性讲义。

## 四、学生端功能

### 1. 交互与场景切换（多形态适配）
* **基础信息同步**：系统自动读取账号年级信息（如：七年级），作为底层对话上下文。
* **三形态适配（Mini / 悬浮窗 / 沉浸页）**：
    * **Mini 态**：胶囊状悬浮球，仅展示“语音输入”与“截屏提问”按钮，极简不遮挡屏幕，适合网课挂机/阅读场景。
    * **悬浮窗态（Side Bar）**：侧边展开的对话窗口。适合日常轻量级答疑，不打断主任务流程。
    * **沉浸页（Full Webpage）**：**独立的全屏网页端**。拥有完整的仪表盘视图，适合进行长时间的“考前冲刺”模拟考、查看复杂的“学习总览”图表或进行深度错题复盘，提供无干扰的专注环境。
* **屏幕上下文感知（一键截屏）**：
    * 在 Mini/悬浮窗模式下，提供**“选取截屏/读取页面”**按钮。
    * 点击后支持框选当前屏幕区域（或自动解析当前网页文本），直接作为提问背景发送给 AI，省去“截图-保存-上传”步骤。
* **频道式切换设计**：
    * **默认状态（随便问问）**：全能助手频道，支持多学科混杂提问、百科问答，AI 使用通用模型回复。
    * **一键切换（专项频道）**：点击“数学”、“英语”等图标，进入**专项深度模式**，加载学科专属 System Prompt（如数学老师的逻辑引导），并强制进行知识点打标。

### 2. 智能答疑与解析
* **多模态交互**：支持语音 / 文字 / 屏幕截图 / 文件上传 实时提问。
* **引导式解析**：AI 识别内容后，不直接给出答案，而是通过多轮对话引导学生思考解题步骤。
* **思路笔记**：答疑结束后，自动生成轻量化思路笔记卡片。

### 3. 难点攻克与考前冲刺
* **难点看板**：自动统计并汇总个人高频难点，展示为“待攻克”面板。
* **自主靶向出题**：手动选择难点，AI 生成针对性练习（难度：基础/进阶/综合）。
* **“考前冲刺”模式（Exam Sprint）**：
    * **一键生成**：基于学生过去一周/一月的错题记录与高频薄弱点，AI 智能组合生成一张**“定制冲刺卷”**（包含 5-10 道高价值题目）。
    * **建议场景**：推荐在**“沉浸页”**形态下打开，以获得接近真实考试的体验。

### 4. 进度与反馈（游戏化机制）
* **“错题消消乐”机制 (Gamified Review)**：
    * **触发场景**：在“随便问问”或空闲时段，AI 主动发起互动。
    * **玩法**：弹出 1-3 道历史错题的变式题，学生答对一题，“难点看板”中的对应薄弱点即消除/点亮一颗星。
    * **激励**：通过快速反馈消除“待办难点”的焦虑感，增加复习趣味性。
* **学习总览**：展示已解决难点、待解决痛点、知识点掌握度雷达图。

### 5. 数据权限
* 仅展示本人学习数据，无法查看其他学生信息。
* 支持手机、平板、电脑多端数据同步。
