# 云端项目共用库

## 核心主干项目 (InteractiveDisplay)

该仓库是互动大屏的入口主应用（Vite + React）。

**克隆仓库：**

```bash
git clone https://codeup.aliyun.com/5f0e5cdb769820a3e817ddd9/chanpin/InteractiveDisplay.git
```

## 关联子模块依赖 (运行要求)

> [!IMPORTANT]
> `InteractiveDisplay` 中的 **"AI生漫画"** 模块采用了 Iframe 无缝嵌入架构。
> 本地开发或私有化部署时，**必须同时运行**原生的 `ai-comic-factory` 项目。

**运行要求：**

1. 确保 `ai-comic-factory` 在本地 `3000` 端口运行 (`npm run dev`)。
2. `InteractiveDisplay` 所在的地址（如 `localhost:5000`）通过跨域通信 (`postMessage`) 与子窗口进行路由联动。
