## 登录认证详细流程

```mermaid
flowchart TD
    Start([打开应用]) --> InputCred[输入用户名/密码]
    InputCred --> IsAdmin{是 admin?}

    IsAdmin --> |是| AdminLogin[内置 admin 校验<br/>密码: 123456]
    IsAdmin --> |否| DBQuery[查询 users 表]

    DBQuery --> QueryAPI[GET /data/injhuCnRig<br/>filter: name = 输入值]
    QueryAPI --> Found{找到用户?}

    Found --> |否| LoginFail[登录失败<br/>用户不存在]
    Found --> |是| CheckPwd{密码匹配?}

    CheckPwd --> |否| PwdFail[登录失败<br/>密码错误]
    CheckPwd --> |是| SetSession[存储登录态]

    AdminLogin --> CheckAdminPwd{密码正确?}
    CheckAdminPwd --> |否| PwdFail
    CheckAdminPwd --> |是| SetSession

    SetSession --> SaveUser[保存 currentUser]
    SaveUser --> SetBodyClass[设置 body.is-admin]
    SetBodyClass --> EnterApp[进入应用]

    EnterApp --> CheckRole{角色判断}
    CheckRole --> |admin| HideSidebar[隐藏业务菜单<br/>仅显示版本管理]
    CheckRole --> |staff| ShowAll[显示全部菜单]

    HideSidebar --> CheckUpdate[静默检查更新]
    ShowAll --> CheckUpdate
    CheckUpdate --> Ready([应用就绪])

    style Start fill:#e1f5e1
    style Ready fill:#e1ffe1
    style LoginFail fill:#ffe1e1
    style PwdFail fill:#ffe1e1
```

## API 调用时序图

```mermaid
sequenceDiagram
    participant U as 用户
    participant E as Electron 渲染进程
    participant M as Electron 主进程
    participant API as BaseMulti API
    participant FS as 本地文件系统
    participant OSS as 文件存储服务

    Note over U,OSS: ── 登录流程 ──

    U->>E: 输入用户名/密码
    E->>API: GET /data/injhuCnRig<br/>filter: name = 用户名
    API-->>E: 返回用户记录
    E->>E: 校验密码 & 存储登录态
    E->>E: 渲染对应视图

    Note over U,OSS: ── 数据 CRUD（以任务为例）──

    U->>E: 创建任务
    E->>API: POST /data/SxqN1eraUE<br/>x-bm-token 认证
    API-->>E: 返回新记录 ID
    E->>E: 刷新任务列表

    U->>E: 上传附件
    E->>OSS: POST uploadUnified<br/>Authorization: Bearer 认证
    OSS-->>E: 返回文件 URL
    E->>API: PATCH /data/SxqN1eraUE/:id<br/>更新 files 字段
    API-->>E: 更新成功

    Note over U,OSS: ── Claude Code 切换 ──

    U->>E: 选择 Claude 账号
    E->>M: IPC: writeClaudeSettings
    M->>FS: 写入 ~/.claude/settings.json
    FS-->>M: 写入成功
    M-->>E: 返回成功
    E->>E: 刷新账号列表

    Note over U,OSS: ── 版本更新 ──

    E->>API: GET /data/versions<br/>查询最新版本
    API-->>E: 返回版本列表
    E->>E: 对比本地版本号

    alt 有新版本
        E->>M: IPC: downloadAndInstallUpdate
        M->>M: 后台下载 EXE
        M->>M: 执行 /S 静默安装
        M-->>E: 更新完成
    end
```

## 任务管理详细流程

```mermaid
flowchart TD
    Start([进入任务管理]) --> ViewSwitch{视图切换}

    ViewSwitch --> |全局| GlobalView[全局任务列表]
    ViewSwitch --> |个人| PersonalView[我的任务列表]

    GlobalView --> Filter[筛选/搜索]
    PersonalView --> Filter

    Filter --> TaskList[任务卡片列表]
    TaskList --> SelectTask{操作选择}

    SelectTask --> |新建| CreateTask[创建任务]
    SelectTask --> |查看| TaskDetail[任务详情弹窗]
    SelectTask --> |编辑| EditTask[编辑任务]
    SelectTask --> |删除| DeleteTask[删除任务]

    CreateTask --> SetTitle[设置标题]
    SetTitle --> SetType[选择类型<br/>主线/支线]
    SetType --> SetPriority[设置优先级<br/>高/中/低]
    SetPriority --> SetDeadline[设置截止日期]
    SetDeadline --> SetAssignees[分配执行人]
    SetAssignees --> SetDesc[填写描述]
    SetDesc --> UploadFiles{上传附件?}

    UploadFiles --> |是| Upload[上传到 OSS]
    UploadFiles --> |否| SaveTask[保存任务]
    Upload --> SaveTask

    SaveTask --> PostAPI[POST /data/SxqN1eraUE]
    PostAPI --> SendNotify[发送站内通知<br/>给执行人]
    SendNotify --> RefreshList[刷新列表]

    TaskDetail --> ViewFiles[查看附件]
    ViewFiles --> PreviewImg[图片预览 Lightbox]
    ViewFiles --> DownloadFile[下载文件]

    TaskDetail --> CreateSubTask[创建子任务]
    CreateSubTask --> SetParentId[设置 parent_task_id]
    SetParentId --> SaveTask

    TaskDetail --> UpdateStatus[更新状态]
    UpdateStatus --> StatusFlow{状态流转}
    StatusFlow --> S1[待接取]
    StatusFlow --> S2[进行中]
    StatusFlow --> S3[待验收]
    StatusFlow --> S4[已完成]

    RefreshList --> End([完成])

    style Start fill:#e1f5e1
    style End fill:#e1e5ff
    style CreateTask fill:#fff4e1
    style TaskDetail fill:#e1f5ff
```
