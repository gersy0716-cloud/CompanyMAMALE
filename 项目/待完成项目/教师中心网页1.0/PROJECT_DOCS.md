## 1. 项目简介

**教师中心**是一个专为教育工作者设计的智能体工具聚合平台。它旨在通过整合 30+ 个精选 AI 工具，覆盖 AI 教学、办公写作、班级管理、备课教案、成绩管理等 6 大核心场景，提升教师的工作效率。

### 核心价值
*   **一站式导航**：聚合分散的 AI 工具，提供统一入口。
*   **视觉愉悦**：采用清透、柔光的玻璃拟态设计，缓解视觉疲劳。
*   **交互高效**：自适应便当盒 (Bento) 布局与智能预览，信息获取更直接。

## 2. 核心功能与亮点

### 2.1 🍱 智能 Bento 布局系统
采用独创的 **Unit-Sum Packing** 算法，解决了传统网格布局“留白”或“对不齐”的问题：
*   **自动填充**：算法实时计算每一行的剩余空间。
*   **智能拉伸**：当行末空间不足以容纳标准卡片时，自动调整前一个或最后一个卡片的尺寸（横跨 2 列），确保视觉上的“零空隙”。
*   **多端适配**：
    *   **PC 端**：4 列标准布局，支持 2x2（重点）和 1x1（普通）规格。
    *   **平板**：3 列布局，自动降级重点卡片尺寸。
    *   **手机**：**Micro Bento**（2 列）布局，极致紧凑。

### 2.2 📱 移动端极致适配
*   **Micro Bento**：在手机上重构了网格系统，保持了大小错落的设计感，而非枯燥的单列列表。
*   **视觉优化**：增加了板块间距 (40px) 和网格间隙 (12px)，提升移动端阅读呼吸感。
*   **极简体验**：自动隐藏预览浮窗和非核心导航元素，聚焦内容展示。
*   **横向滑块**：“最近使用”区域在手机上自动转换为横向滚动，节省垂直空间。

### 2.3 🔍 高效交互
*   **悬停预览**：PC 端鼠标悬停在工具卡片上时，侧边自动弹出大尺寸截图预览（320x180px）。
*   **快速导航**：顶部导航栏集成锚点跳转，点击即可平滑滚动至对应板块。
*   **全局搜索**：顶部居中的搜索框，支持实时筛选工具。

## 3. 技术架构

### 3.1 技术栈
*   **HTML5**：语义化标签结构。
*   **CSS3**：
    *   **Grid & Flexbox**：核心布局引擎。
    *   **CSS Variables**：统一管理色板、圆角和阴影。
    *   **Backdrop Filter**：实现玻璃拟态效果。
*   **Vanilla JavaScript (ES6+)**：无依赖的原生 JS 实现，轻量高效。

## 4. 维护指南

### 4.1 如何添加新工具？
打开 `data.js`，在对应的 `tools` 数组中添加对象：

```javascript
{
    id: 'unique-id',
    name: '工具名称',
    icon: '图标(SVG或Emoji)',
    description: '简短描述',
    url: '跳转链接',
    featured: true // 是否为重点推荐 (PC端占大格)
}
```

### 4.2 如何调整布局列数？
(仅限高级修改) 修改 `app.js` 中的 `BentoLayoutManager.getCurrentColumns()` 方法：

```javascript
static getCurrentColumns() {
    const width = window.innerWidth;
    if (width > 1200) return 4;
    if (width > 700) return 3; // 平板
    return 2; // 手机
}
```

### 4.3 常见问题
*   **Q: 预览浮框在手机上不显示？**
    *   A: 设计如此。为了保证移动端阅读体验，CSS 中通过 `@media` 查询隐藏了 `.tool-preview-card`。
*   **Q: 卡片为什么会自动变宽？**
    *   A: 这是智能填充算法在工作，为了填满行末的空隙，CSS `flex-grow` 和 JS 动态分配的 `size-medium` 会共同作用。
