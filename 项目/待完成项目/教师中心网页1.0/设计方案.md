## 📋 项目概述

### 功能定位
教师中心是一个智能体工具集合平台，按教师工作场景分类展示各类AI工具，帮助教师快速找到并使用所需的智能体。

### 核心功能
1. **6大板块分类展示**：AI教学、办公写作工具、班级管理、备课教案、成绩与学业管理、课堂工具
2. **智能体工具展示**：每个板块下展示多个智能体小工具（当前约50个）
3. **AI助手对话**：悬浮AI助手，帮助教师找到需要的智能体
4. **可扩展性**：支持动态增加板块和智能体

## 📐 技术实现方案

### 1. 自适应网格布局

```css
.grid-container {
    display: grid;
    /* 自动填充，每列最小300px，最大1fr */
    grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
    /* 行高自动适应内容 */
    grid-auto-rows: auto;
    gap: 20px;
    padding: 20px;
}

/* 板块卡片 - 高度由内容决定 */
.block-card {
    /* 无需设置固定高度 */
    display: flex;
    flex-direction: column;
}

/* 智能体网格 - 自动换行 */
.tools-grid {
    display: grid;
    grid-template-columns: repeat(auto-fill, minmax(100px, 1fr));
    gap: 12px;
}
```

**说明**：
- 使用 `auto-fit` 让列数自动适应容器宽度
- 使用 `auto` 让行高根据内容自动调整
- 智能体网格使用 `auto-fill` 自动换行
- 完全不需要手动设置板块大小

### 2. 主色调提取

```javascript
/**
 * 从图片提取主色调
 * @param {string} imageUrl - 图片URL
 * @returns {string} 浅色版主色调（hex格式）
 */
async function extractMainColor(imageUrl) {
    const img = new Image();
    img.crossOrigin = 'Anonymous';
    img.src = imageUrl;

    await img.decode();

    const canvas = document.createElement('canvas');
    const ctx = canvas.getContext('2d');
    canvas.width = img.width;
    canvas.height = img.height;
    ctx.drawImage(img, 0, 0);

    // 提取主色调逻辑
    const imageData = ctx.getImageData(0, 0, canvas.width, canvas.height);
    const mainColor = getMainColorFromImageData(imageData);

    // 转换为浅色版本
    return lightenColor(mainColor, 0.9);
}
```

## 🔄 可扩展性设计

### 1. 动态增加板块
- 在数据数组中添加新板块对象
- Grid 布局自动调整列数和行数
- 板块高度根据工具数量自动增长
- 无需修改任何 CSS

### 2. 动态增加智能体
- 在对应板块的 `tools` 数组中添加
- 自动提取主色调
- 自动生成卡片
- 板块高度自动扩展

### 3. 完全自适应
- 无需设置板块大小
- 工具数量决定板块高度
- 列数根据屏幕宽度自动调整
- 新增内容自动排列

## 📱 响应式适配

### 桌面端（> 1200px）
- 自动填充3-4列（根据容器宽度）
- 板块高度根据内容自动调整
- 悬停预览效果

### 平板端（768px - 1200px）
- 自动填充2列
- 板块高度自适应
- 保留悬停效果

### 手机端（< 768px）
- 单列布局
- 板块高度自适应
- 点击查看预览（替代悬停）
