# 教师中心网页 1.0

这是一个专为教师设计的智能体工具集合平台，包含 AI 教学、办公写作、班级管理等 6 大板块，汇集了 30+ 个实用 AI 工具。

## ✨ 核心特性

- **Bento 布局 (便当盒)**：自适应网格系统，采用 Unit-Sum Packing 算法，确保卡片完美填充无空隙。
- **微型移动端适配**：在手机上自动切换为 Micro Bento 布局（双列），保持设计感的同时适应小屏。
- **动态预览**：悬停工具时展示大尺寸截图预览。
- **快速导航**：顶部导航栏集成搜索与快速跳转锚点。
- **视觉风格**：清透、柔光、玻璃拟态 UI，专为教师打造的舒适视觉体验。

## 🛠 技术栈

- **HTML5 / CSS3 (Grid & Flexbox)**
- **Vanilla JavaScript (ES6+)**
- **SVG 图标系统**

## 📐 布局算法说明

项目使用自定义的 `BentoLayoutManager` 进行网格计算：
1.  **基础网格**：将页面分为 4 列 (PC)、3 列 (平板) 或 2 列 (手机)。
2.  **尺寸分配**：
    -   **Large (2x2)**: 重点推荐工具。
    -   **Medium (2x1)**: 用于补齐行末空缺。
    -   **Small (1x1)**: 标准工具卡片。
3.  **智能填充**：算法会自动检测每一行的剩余空间 (`remainder`)，动态调整最后一个元素的尺寸 (`flex-grow` 或 `grid-column-end`)，确保视觉上永远填满，杜绝网格缺口。

直接在浏览器中打开 `index.html` 即可预览效果。无需构建步骤。
