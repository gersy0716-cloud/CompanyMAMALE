## URL参数说明

> ⚠️ **重要规则**：
>
> - ❌ 不能在代码中默认设置URL参数
> - ✅ URL参数必须由用户在地址栏中提供
> - ⚠️ 没有URL参数则无权限访问

### 参数列表

| 参数名 | 说明 | 格式 |
|:-------|:-----|:-----|
| `type` | 租户英文名称 | 例如：3w-api |
| `tenant` | 租户ID | UUID格式 |
| `author` | 用户中文名 | URL编码（可选） |
| `userid` | 用户在该租户下的ID | UUID格式 |
| `username` | 用户中文名 | URL编码 |
| `token` | 学生访问令牌 | JWT格式 |
| `teachertoken` | 教师令牌 | JWT格式（教师端必需） |

### 模式判断逻辑（app.js）

```javascript
// 解析URL参数
function parseURLParams() {
    const urlParams = new URLSearchParams(window.location.search);
    return {
        type: urlParams.get('type'),
        tenant: urlParams.get('tenant'),
        userid: urlParams.get('userid'),
        username: decodeURIComponent(urlParams.get('username') || ''),
        token: urlParams.get('token'),
        teachertoken: urlParams.get('teachertoken')
    };
}

// 判断用户模式
const params = parseURLParams();
const isTeacher = !!params.teachertoken;

if (isTeacher) {
    // 教师模式 - 显示管理界面
    await initTeacherView();
} else {
    // 学生模式 - 显示查看界面
    await initStudentView();
}
```

## API接口路径

### 外部API（mamale.vip）

用于获取班级和学生数据：

| 接口 | URL | 方法 | 认证 |
|------|-----|------|------|
| 班级列表 | `https://{type}.mamale.vip/api/app/class` | GET | Bearer teachertoken |
| 班级学生 | `https://{type}.mamale.vip/api/app/student/byClssId?ClassId={id}` | GET | Bearer teachertoken |
| 学生信息 | `https://{type}.mamale.vip/api/app/student/studentInfo` | GET | Bearer token |

#### 请求头示例

```javascript
const headers = {
    'Authorization': `Bearer ${token}`
};
```

#### 分页参数

```javascript
// 支持分页查询
const url = `${baseUrl}?PageSize=100&PageIndex=1`;
```

### BaseMulti数据库API

用于存储值日数据：

| 操作 | 方法 | URL |
|------|------|-----|
| 查询记录 | GET | `/api/bases/{base_id}/tables/{table_id}/records` |
| 创建记录 | POST | `/api/bases/{base_id}/tables/{table_id}/records` |
| 更新记录 | PATCH | `/api/bases/{base_id}/tables/{table_id}/records/{record_id}` |
| 删除记录 | DELETE | `/api/bases/{base_id}/tables/{table_id}/records/{record_id}` |

#### 请求头示例

```javascript
const headers = {
    'Authorization': `Bearer ${DB_CONFIG.apiKey}`,
    'Content-Type': 'application/json'
};
```

## 重要配置注意事项

### 1. BaseMulti特殊字段类型

- **Switch字段**：返回数字 `1` 或 `0`，不是布尔值
- **JSON字段**：必须使用 `long text` 类型，存储为字符串格式
- **Date字段**：格式为 `YYYY-MM-DD`

### 2. 日期处理

```javascript
// ✅ 正确：使用本地日期格式
const formatDate = (date) => {
    const year = date.getFullYear();
    const month = String(date.getMonth() + 1).padStart(2, '0');
    const day = String(date.getDate()).padStart(2, '0');
    return `${year}-${month}-${day}`;
};

// ❌ 错误：使用toISOString()会导致时区问题
const wrongDate = new Date().toISOString().split('T')[0];
```

### 3. ID比较

```javascript
// ✅ 使用宽松比较（因为ID可能是字符串或数字）
const group = groups.find(g => g.id == groupId);

// ❌ 避免使用严格比较
const group = groups.find(g => g.id === groupId); // 可能匹配失败
```

### 4. JSON数据处理

```javascript
// 保存时转换为字符串
const data = {
    student_ids: JSON.stringify(['张三', '李四'])
};

// 读取时解析为数组
const studentIds = JSON.parse(assignment.student_ids || '[]');
```

VITE_DB_BASE_ID=your_base_id

在代码中使用：

```javascript
const apiUrl = import.meta.env.VITE_DB_API_URL;
```
