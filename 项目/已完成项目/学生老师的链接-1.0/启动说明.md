# 项目启动说明

## ⚠️ CORS问题解决方案

根据`CLAUDE.md`要求:**必须通过http://localhost访问，不能用file://协议**

---

## 🚀 启动步骤

### 方法1: Python HTTP服务器 (推荐)

1. 打开PowerShell或命令提示符
2. 进入项目目录:
```powershell
cd "G:\工作\码码乐\工作—码码乐-开发\待完成项目\学生老师的链接-1.0"
```

3. 启动HTTP服务器:
```powershell
python -m http.server 8000
```

4. 浏览器访问:
```
http://localhost:8000/test-launcher.html
```

5. 填写参数后点击"启动系统"

---

### 方法2: VS Code Live Server

1. 在VS Code中打开项目文件夹
2. 安装"Live Server"扩展(如果还没安装)
3. 右键点击`test-launcher.html`
4. 选择"Open with Live Server"
