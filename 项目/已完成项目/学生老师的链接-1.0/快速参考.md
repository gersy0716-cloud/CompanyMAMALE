## 🔧 常见技术问题

### Electron Builder 符号链接权限错误

**症状**: `ERROR: Cannot create symbolic link : 客户端没有所需的特权`

**快速解决**:
```powershell
# 方案1: 启用开发者模式（设置 → 开发者选项）
# 方案2: 清除缓存
Remove-Item -Recurse -Force "$env:LOCALAPPDATA\electron-builder\Cache\winCodeSign"
# 方案3: 管理员运行
```
