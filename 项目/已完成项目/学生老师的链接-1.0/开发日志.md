### 💡 改进建议

#### 1. 添加数据缓存

**目的**: 减少重复API请求，提升用户体验

**实现思路**:
```javascript
const cache = {
  classes: { data: null, timestamp: null },
  students: {}, // key: classId, value: { data, timestamp }
  teachers: { data: null, timestamp: null }
};

const CACHE_DURATION = 5 * 60 * 1000; // 5分钟

function getCachedData(key) {
  const cached = cache[key];
  if (!cached || !cached.timestamp) return null;
  
  const now = Date.now();
  if (now - cached.timestamp > CACHE_DURATION) {
    return null; // 缓存过期
  }
  
  return cached.data;
}
```

## 相关技术问题记录

### Electron Builder 权限问题（记账项目）

**问题**: 构建 Electron 应用时出现符号链接权限错误

```
ERROR: Cannot create symbolic link : 客户端没有所需的特权
```

**原因**: Windows 默认需要管理员权限才能创建符号链接

**解决方案**:
1. ✅ **启用 Windows 开发者模式**（推荐）
   - 设置 → 隐私和安全性 → 开发者选项 → 开启开发人员模式
   - 或使用 PowerShell（管理员）：
     ```powershell
     reg add "HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\AppModelUnlock" /t REG_DWORD /f /v "AllowDevelopmentWithoutDevLicense" /d "1"
     ```

2. 以管理员身份运行构建命令
3. 清除缓存：`Remove-Item -Recurse -Force "$env:LOCALAPPDATA\electron-builder\Cache\winCodeSign"`
4. 在 `package.json` 中禁用代码签名：`"win": { "sign": null }`
