## 图片生成详细流程

```mermaid
flowchart TD
    Start([开始图片生成]) --> BuildPrompt[构建视觉提示词]
    BuildPrompt --> Combine[组合标题+内容+风格]
    Combine --> CreateTask[创建异步任务]

    CreateTask --> PostAPI[POST asyncImageCreateMy]
    PostAPI --> GetTaskId[获取TaskId]

    GetTaskId --> Poll[轮询查询]
    Poll --> PostQuery[POST asyncImageQueryMy]
    PostQuery --> CheckStatus{检查状态}

    CheckStatus --> |pending| Wait[等待2秒]
    CheckStatus --> |processing| Wait
    Wait --> PollCount{轮询次数<30?}
    PollCount --> |是| Poll
    PollCount --> |否| Timeout[超时失败]

    CheckStatus --> |completed| GetURL[获取图片URL]
    CheckStatus --> |failed| Error[生成失败]

    GetURL --> ConvertURL[转换URL到阿里云]
    ConvertURL --> PostConvert[POST myConvertUrl]
    PostConvert --> GetAliyunURL[获取阿里云URL]
    GetAliyunURL --> SaveURL[保存到slide.imageUrl]
    SaveURL --> Success([生成成功])

    Timeout --> End([结束])
    Error --> End

    style Start fill:#e1f5e1
    style Success fill:#e1ffe1
    style End fill:#ffe1e1
    style Timeout fill:#ffcccc
    style Error fill:#ffcccc
```

## 云端保存详细流程

```mermaid
flowchart TD
    Start([开始保存]) --> CheckMode{编辑模式?}

    CheckMode --> |new| NewMode[新建模式]
    CheckMode --> |edit| EditMode[编辑模式]
    CheckMode --> |view| ViewMode[查看模式<br/>另存为新课件]

    NewMode --> SelectCat[选择课件分类]
    EditMode --> SelectCat
    ViewMode --> SelectCat

    SelectCat --> InputName[输入课件名称]
    InputName --> SetShare[设置是否共享]

    SetShare --> GenJSON[生成JSON数据]
    GenJSON --> IncludeSlides[包含slides数组]
    GenJSON --> IncludeSettings[包含settings配置]

    IncludeSlides --> UploadJSON[上传JSON到OSS]
    IncludeSettings --> UploadJSON

    UploadJSON --> PostUpload[POST uploadUnified]
    PostUpload --> GetJSONURL[获取JSON文件URL]

    GetJSONURL --> GenCover[生成封面图]
    GenCover --> Screenshot[截取第一页]
    Screenshot --> UploadCover[上传封面图]
    UploadCover --> GetCoverURL[获取封面URL]

    GetCoverURL --> PrepareData[准备保存数据]
    PrepareData --> SetCategoryId[aiApplicationCategoryId]
    PrepareData --> SetName[name]
    PrepareData --> SetCoverImg[converImg]
    PrepareData --> SetPPTData[pptData]
    PrepareData --> SetIsShare[isShare]

    SetCategoryId --> SaveAPI{保存API}
    SetName --> SaveAPI
    SetCoverImg --> SaveAPI
    SetPPTData --> SaveAPI
    SetIsShare --> SaveAPI

    SaveAPI --> |新建| PostCreate[POST /app/aiPPTX/my]
    SaveAPI --> |更新| PutUpdate[PUT /app/aiPPTX/:id/my]

    PostCreate --> SaveDB[保存到数据库]
    PutUpdate --> SaveDB

    SaveDB --> InsertRecord[插入/更新记录]
    InsertRecord --> SetCreator[设置创建者信息]
    InsertRecord --> SetTenant[设置租户ID]
    InsertRecord --> SetTimestamp[设置时间戳]

    SetCreator --> Success[保存成功]
    SetTenant --> Success
    SetTimestamp --> Success

    Success --> BackCenter[返回课件中心]
    BackCenter --> End([结束])

    style Start fill:#e1f5e1
    style Success fill:#e1ffe1
    style End fill:#e1e5ff
```

## API调用时序图

```mermaid
sequenceDiagram
    participant U as 用户
    participant F as 前端应用
    participant API as 后端API
    participant AI as AI服务
    participant OSS as 对象存储
    participant DB as 数据库

    U->>F: 输入教案内容
    F->>F: 配置生成选项
    U->>F: 点击"生成大纲"

    F->>API: POST OpenAI Stream API
    API->>AI: 调用AI模型
    AI-->>API: 流式返回大纲
    API-->>F: SSE流式传输
    F->>F: 实时显示大纲

    U->>F: 编辑大纲
    U->>F: 点击"批量生成图片"

    loop 每个幻灯片
        F->>API: POST asyncImageCreateMy
        API->>AI: 创建图片生成任务
        AI-->>API: 返回TaskId
        API-->>F: 返回TaskId

        loop 轮询查询
            F->>API: POST asyncImageQueryMy
            API->>AI: 查询任务状态
            AI-->>API: 返回状态和URL
            API-->>F: 返回结果
        end

        F->>API: POST myConvertUrl
        API->>OSS: 转存图片到阿里云
        OSS-->>API: 返回阿里云URL
        API-->>F: 返回阿里云URL
        F->>F: 保存图片URL
    end

    U->>F: 点击"保存到云端"
    F->>F: 生成JSON数据

    F->>API: POST uploadUnified
    API->>OSS: 上传JSON文件
    OSS-->>API: 返回文件URL
    API-->>F: 返回JSON URL

    F->>F: 生成封面图
    F->>API: POST uploadUnified
    API->>OSS: 上传封面图
    OSS-->>API: 返回封面URL
    API-->>F: 返回封面URL

    F->>API: POST /app/aiPPTX/my
    API->>DB: 插入课件记录
    DB-->>API: 返回课件ID
    API-->>F: 保存成功

    F->>U: 显示成功提示
    F->>F: 返回课件中心
```
