# AI备课工具 - 功能与特点

## 项目概述

这是一个基于AI的智能PPT课件生成工具，专为中小学教师设计，能够自动将教案、课文等教学材料转换为专业的多媒体课件。

---

## 一、核心功能模块

### 1. 课件生成流程

#### 1.1 输入阶段
**文件位置**: `components/InputPanel.tsx`

**支持的输入方式**:
- 直接粘贴文本内容
- 上传 .txt 或 .md 文件
- 支持 Markdown 格式

**配置选项**:
- **预计页数**: 3-50页可调（滑块控制）
- **学科分类**: 14个学科
  - 语文、数学、英语、物理、化学、生物
  - 地理、历史、道德与法治
  - 信息技术、美术、音乐、科学、教师论文
- **课件风格**: 35种预设风格 + 自定义风格
- **内容侧重**: 概括摘要、详细内容、视觉为主
- **语音合成**: 可选启用（为每页生成语音文稿）

#### 1.2 大纲编辑阶段
**文件位置**: `components/OutlineEditor.tsx`

**功能特点**:
- AI自动生成课件大纲结构
- 拖拽排序（使用 @hello-pangea/dnd）
- 实时编辑标题、内容、视觉提示词
- 编辑语音文稿（如果启用）
- 添加/删除页面
- 显示字数统计

#### 1.3 工作台阶段
**文件位置**: `components/WorkbenchStage.tsx`

这是最核心的编辑界面，包含4个主要TAB：
- TAB 1: 图片生成与编辑
- TAB 2: 语音合成
- TAB 3: 视频生成
- TAB 4: 互动工具

---

### 2. 图片生成功能（TAB 1）

#### 2.1 AI图片生成
**服务文件**: `services/geminiService.ts`

**技术特点**:
- 使用统一的 OpenAI Stream API
- 支持 SSE（Server-Sent Events）流式传输
- 可配置通道（Provider）和模型
- 默认模型: `nano-banana-2-vip`
- 默认通道: `TuZi`
- 超时控制: 150秒
- 支持中止生成（AbortController）

**生成模式**:
- **批量生成**: 一键生成所有页面图片
- **单页生成**: 为当前页生成图片
- **重新生成**: 重新生成当前页图片

#### 2.2 图片编辑功能
**文件位置**: `components/ImageEditor.tsx`

**编辑工具**:
- **画笔工具**: 红色标注（可调节笔刷大小 1-20）
- **橡皮擦工具**: 擦除标注恢复原图
- **清除功能**: 一键清除所有标注
- **图生图**: 上传编辑后的图片，AI根据标注重新生成

**工作流程**:
1. 在Canvas上绘制标注
2. 输入修改提示词
3. 将Canvas转为Blob
4. 上传到服务器获取URL
5. 调用图生图API重新生成

---

### 3. 语音合成功能（TAB 2）

**服务文件**: `services/ttsService.ts`

**技术实现**:
- 使用火山引擎 TTS API
- 异步任务模式（提交任务 → 轮询查询）
- 支持 MP3 格式
- 最大轮询次数: 30次
- 轮询间隔: 2秒

**功能特点**:
- 为每页单独生成语音
- 实时编辑语音文稿
- 字数统计（建议200字以内）
- 语音试听（播放/暂停）
- 重新生成语音

---

### 4. 视频生成功能（TAB 3）

**服务文件**: `services/videoGenerationService.ts`

**技术实现**:
- 使用 Sora-2 模型
- 图片转视频（Image-to-Video）
- 异步任务模式（创建任务 → 轮询查询）
- 最大轮询次数: 100次
- 轮询间隔: 3秒

**工作流程**:
1. 压缩图片到1024宽度
2. 上传压缩后的图片
3. 创建视频生成任务（提供提示词）
4. 轮询等待生成完成（60-90秒）
5. 获取视频URL

**视频上传**:
- 支持上传本地视频
- 文件大小限制: 10MB
- 支持常见视频格式（MP4、MOV、AVI等）

---

### 5. 互动工具功能（TAB 4）

**数据结构**: `types.ts`

**功能特点**:
- 在幻灯片上添加可点击的交互点
- 每个交互点包含HTML内容
- 支持拖拽定位（百分比坐标）
- 自动编号标签（1, 2, 3...）
- 可配置弹窗尺寸（宽度/高度百分比）

**HTML内容生成**:
- 手动输入HTML代码
- AI自动生成HTML（提供提示词和背景色）
- 实时预览
- 可调整弹窗大小（拖拽四角）

**交互点样式**:
- 圆点（dot）
- 图钉（pin）
- 星星（star）
- 点赞（thumbsUp）
- 点踩（thumbsDown）

---

### 6. 绘图标注功能

**工具类型**:
- **画笔**: 自由绘制
- **箭头**: 绘制指向箭头
- **矩形**: 绘制矩形框
- **热点**: 添加交互热点

**配置选项**:
- 颜色选择器（默认红色 #EF4444）
- 笔刷大小调节
- 橡皮擦工具
- 清除所有标注

## 四、导出功能

### 1. PPTX导出
**服务文件**: `services/pptxService.ts`

**技术实现**:
- 使用 pptxgenjs 库
- 16:9 宽高比
- 图片作为背景铺满
- 自动转换图片为Base64
- 失败页面显示文本内容

### 2. PDF导出
**服务文件**: `services/pdfService.ts`

**技术实现**:
- 使用 jsPDF 库
- A4横向格式
- 图片自动缩放适配

### 3. JSON导出/导入
**服务文件**: `services/jsonService.ts`

**功能**:
- 导出完整课件数据（slides + settings）
- 导入JSON恢复课件
- 用于本地保存和分享

## 五、课件中心功能

**文件位置**: `components/CoursewareCenter.tsx`

### 三个TAB视图
1. **所有课件**: 显示平台所有课件
2. **本校区课件**: 根据 TenantId 筛选本校课件
3. **我的课件**: 仅显示当前用户创建的课件

### 功能特点
- 分页显示（每页20个）
- 搜索功能（按课件名称）
- 分类筛选（按学科分类）
- 课件卡片显示：
  - 封面图
  - 课件名称
  - 创建时间
  - 页数统计
  - 创建者信息
  - 浏览次数

### 操作功能
- **查看**: 只读模式，保存时另存为新课件
- **编辑**: 编辑模式，保存时更新原课件
- **删除**: 删除我的课件
- **生成新课件**: 跳转到输入页面

## 六、云端保存功能

**服务文件**: `services/cloudSaveService.ts`

### 保存流程
1. 选择课件分类（学科）
2. 输入课件名称
3. 选择是否共享
4. 上传JSON数据文件到OSS
5. 生成封面图（第一页截图）
6. 保存课件元数据到服务器

### 编辑模式
- **新建模式** (new): 创建新课件
- **编辑模式** (edit): 更新现有课件
- **查看模式** (view): 只读，保存时另存为
