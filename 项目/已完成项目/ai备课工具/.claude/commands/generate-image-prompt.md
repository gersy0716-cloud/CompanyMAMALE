# 生成图片提示词

根据用户输入的文稿内容，生成适合 nanobanana 模型的图片生成提示词。

## 使用方法

用户提供文稿内容后，调用 AI 接口生成专业的图片提示词。

## 执行步骤

1. 获取用户输入的文稿内容：$ARGUMENTS

2. 调用以下 API 生成提示词：

```
API 地址: https://dalu.chatgptten.com/v1/chat/completions
模型: gemini-3-pro-preview
API Key: 从环境变量 OPENAI_API_KEY 获取，如果没有则使用默认值
```

3. 使用以下系统提示词：

```
你是一个专业的AI绘画提示词专家，擅长为不同风格生成精确的 nanobanana 模型提示词。
```

4. 使用以下用户提示词模板：

```
请根据以下文稿内容，生成一段详细的 nanobanana 绘画提示词，用于生成 PPT 课件页面。

文稿内容：
{用户输入的文稿}

要求：
1. 描述画面的核心特征（颜色、构图、质感、氛围等）
2. 适合用于 AI 生成 PPT 课件页面
3. 提示词要具体、专业、可执行
4. 必须包含以下要素：
   - 画面主题和内容描述
   - 色彩方案（主色调、配色）
   - 视觉风格（如卡通、写实、扁平化等）
   - 排版布局建议
   - 氛围和情感基调
5. 长度在 150-300 字之间
6. 使用中文
7. 图片比例为 16:9

只输出提示词内容，不要其他说明。
```

5. 将生成的提示词返回给用户，格式如下：

```
## 生成的图片提示词

{生成的提示词内容}

---
提示：你可以使用 /generate-image 命令配合此提示词来生成图片。
```
