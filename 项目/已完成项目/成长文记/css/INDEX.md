# CSS 目录索引

> 样式文件快速索引

## 📁 文件列表

| 文件名 | 说明 | 包含内容 |
|:---|:---|:---|
| `common.css` | 通用样式 | CSS变量、按钮、卡片、输入框、Loading、Toast |
| `mobile.css` | 手机端样式 | 头部、功能卡片、弹窗、安全区域适配 |
| `screen.css` | 屏幕端样式 | TAB导航、轮播、左右分栏、作品列表 |

## 🎨 CSS变量

在 `common.css` 中定义：

- 颜色：`--color-primary`, `--color-success` 等
- 字体：`--font-size-base`, `--font-family`
- 间距：`--spacing-sm`, `--spacing-md`, `--spacing-lg`
- 圆角：`--border-radius-md`, `--border-radius-lg`
- 阴影：`--box-shadow-base`, `--box-shadow-card`
