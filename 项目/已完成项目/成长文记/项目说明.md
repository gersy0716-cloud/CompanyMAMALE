## 📋 项目概述

**项目名称**：成长文记

**项目目标**：为小学语文课堂提供作文教学辅助工具，支持学生作品上传、OCR识别、语音提示词录入、成长时刻展示、AI作文润色等功能。

**使用场景**：五年级上册第一单元习作《那一刻，我长大了》

## 🎯 功能需求

### 手机端（3个功能）

#### 1. 成长时刻上传

- 拍照并上传学生成长瞬间的照片
- 保存到"成长时刻照片表"

#### 2. 作文上传

- 拍摄学生纸质作文
- 调用合合OCR接口识别文字
- 显示识别结果，支持编辑和确认
- 保存照片URL和识别文字到"作品表"

#### 3. 语音提示词

- 录制学生语音提示词（最长60秒）
- 调用火山豆包语音转文字接口
- 保存提示词到"作品表"

### 屏幕端（2个TAB）

#### TAB1：成长时刻

- 轮播展示所有成长时刻照片
- 支持开始/停止/手动刷新
- 本地缓存优化（5分钟有效期）

#### TAB2：作文润色

- 左侧：作品列表、原文显示、提示词输入
- 右侧：AI润色结果（SSE流式输出）
- 支持字体大小调整、图片放大查看

## 🔌 外部API集成

### 1. 文件上传

- **接口**：`POST https://{type}.520ai.cc/api/fileResouceItem/uploadUnified`
- **字段名**：`formFile`

### 2. OCR识别（合合TextIn）

- **接口**：`POST api/app/aiTextin/myPdfToDoMarkdown`
- **参数**：`{ "url": "图片URL", "isToHtml": false }`

### 3. 语音转文字（火山豆包）

- **接口**：`POST api/app/volcengine/audioToTextMy`
- **参数**：`{ "content": "音频URL或base64" }`

### 4. AI对话（基石CoresHub）

- **接口**：`POST api/app/zjAi/myUnifiedOpenAiStream`（SSE流式）
- **模型**：`DeepSeek-V3`

## ⚙️ 配置参数

配置文件：`js/config.js`

| 配置项 | 当前值 | 说明 |
|:---|:---|:---|
| `UPLOAD.MAX_SIZE` | 2MB | 文件上传大小限制 |
| `SLIDESHOW.INTERVAL` | 3000ms | 轮播间隔 |
| `VOICE_RECORDING.MAX_DURATION` | 60s | 最大录音时长 |

## 🏗️ 技术架构

### 技术栈

- **前端**：纯 HTML + CSS + JavaScript（无框架）
- **数据库**：BaseMulti（data.520ai.cc）
- **外部API**：合合OCR + 火山豆包语音 + 基石AI对话

## 📌 注意事项

1. **不区分用户角色**：手机端和屏幕端共享数据
2. **不区分班级**：所有数据在同一个表中
3. **URL参数必需**：必须通过URL传递type、token、userid、username
4. **HTTPS环境**：相机调用需要HTTPS或localhost
